# dekube.io

An entire repository for landing pages? Yep. The horror begins at the first page — nothing is hidden.

Published at **[dekube.io](https://dekube.io/)**.

## Structure

- `index.html` — main landing page
- `privacy.html` — privacy policy (spoiler: there is nothing to disclose)
- `terms.html` — terms & conditions (spoiler: Unlicense)
- `shared.css` — shared styles across all dekube static sites
- `tentacle.js` — the tentacle. It lives in the corner. Do not disturb it.
- `tabs.js` — terminal tab switcher, shared across subpages
